# Security Policy

## Supported Versions

Only the current release is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| current | :white_check_mark: |
| older   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by emailing:

**security@saldestechnology.com**

We ask that you do not publicly disclose the issue until we have had a reasonable opportunity to address it. You can expect an initial response within 5 business days, and we will keep you informed of our progress as we work toward a resolution.

## No Bug Bounty

At this time, we do not offer a bug bounty program.
